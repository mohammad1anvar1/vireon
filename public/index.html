Digital Twin Technology

Every Building Becomes a Living Data Asset

VIREON constructs a full AI-powered energy simulation of your property — modelling thermal performance, building fabric, system efficiency, and occupancy patterns to surface intelligence that standard surveys simply cannot provide.

🏗️

Thermal Envelope Modelling

AI maps heat loss pathways through walls, roofs, windows, and floors using building physics simulation — identifying exactly where energy is escaping.

⚙️

Systems Performance Analysis

Heating, ventilation, hot water, and electrical systems are individually assessed against efficiency benchmarks and current UK standards.

📈

Consumption Pattern Intelligence

Utility data and occupancy patterns are overlaid onto the building model to identify waste, anomalies, and seasonal inefficiencies.

🔄

Continuous Model Refinement

As new data becomes available — new bills, post-retrofit readings, seasonal changes — the model self-updates to keep intelligence current.

Platform Architecture

Built to Scale.
Enterprise-Ready Architecture.

VIREON is architected for national scale and future GCC/European expansion from day one. Every component is modular, API-first, and cloud-native.

🗄️

Database Layer

Supabase (PostgreSQL) for structured data with row-level security. Separate schemas for users, providers, leads, and analytics.

Supabase PostgreSQL RLS
🔐

Auth & User Roles

Multi-role authentication — Customer, Provider, Admin, Enterprise. JWT-based sessions with OAuth2 social login.

Supabase Auth JWT OAuth2
⚡

API Layer

RESTful and GraphQL APIs for all marketplace operations. Webhook system for real-time lead notifications to providers.

Next.js API REST Webhooks
💳

Payments

Stripe for subscription billing, lead fees, and commission processing. Split payments and automated commission distribution.

Stripe Subscriptions Connect
🤖

AI Engine

Property intelligence models running on serverless compute. Lead matching algorithm trained on UK retrofit data and EPC register.

OpenAI API Vercel AI Python ML
📨

CRM & Comms

HubSpot CRM integration for lead management. Transactional email via Resend. SMS notifications via Twilio.

HubSpot Resend Twilio
📊

Analytics

Platform analytics via PostHog. Business intelligence dashboards via Metabase. Real-time metrics via Supabase Realtime.

PostHog Metabase Realtime
☁️

Infrastructure

Vercel for frontend and serverless functions. AWS S3 for document storage. Cloudflare for CDN and DDoS protection.

Vercel AWS S3 Cloudflare
